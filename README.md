# React Jobs Board App

This is the Frontend simple React Jobs Board  application.
The backend part (the server)   can be found here [React Jobs Board Server](https://github.com/Emox87/node-jobs-server)

# Install Dependencies
```bash
npm install
```

### Run Vite Frontend

React will run on http://localhost:3000

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

# Download the server
[React Jobs Board Server](https://github.com/Emox87/node-jobs-server) and install the dependencies
```bash
npm install
```

# Start the Server API
```bash
npm run dev
```
